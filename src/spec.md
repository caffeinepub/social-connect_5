# Specification

## Summary
**Goal:** Build an Instagram-like social media application with user profiles, chat messaging, stories, reels, and a follow system.

**Planned changes:**
- Implement user authentication with Internet Identity and user profiles (username, profile picture, bio)
- Create one-on-one text chat messaging system with conversation history
- Add stories feature with 24-hour expiration for photo content
- Build reels feature for short-form video content with vertical scrolling feed
- Implement follow/unfollow system with follower/following counts and lists
- Design modern social media UI with home feed, navigation, and content sections
- Include app branding and placeholder profile images

**User-visible outcome:** Users can create profiles, follow other users, post and view stories and reels, send private messages, and browse a home feed of content from followed accounts.
